# android-hpu-master
河南理工大学校园通项目
此项目是河南理工大学校园查询客户端.
随时随地查询校园公告，图书借阅，历年成绩，课程表，空闲教室，充值消费等，目前处于内测阶段，一卡通查询只限校园网使用.

注意：本项目最低支持Android2.2
       依赖另一日历选择项目：https://github.com/square/android-times-square
